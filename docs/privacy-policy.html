🔔 Política de Privacidade

Aplicativo Lembre+ — Gerenciamento de Lembretes

Esta Política de Privacidade descreve como o aplicativo Lembre+ ("nós", "nosso" ou "aplicativo") coleta, utiliza e protege as informações dos seus usuários ("você" ou "usuário"). Ao utilizar o Lembre+, você concorda com as práticas descritas neste documento.

Última atualização: 15 de maio de 2026

1. Informações que coletamos

📱 Dados armazenados localmente Offline

O Lembre+ armazena os seguintes dados exclusivamente no seu dispositivo, utilizando um banco de dados local (SQLite):

  • Lembretes criados (nome, descrição, data, categoria, recorrência)
  • Alertas e configurações de notificação
  • Categorias personalizadas
  • Preferências do aplicativo (tema, idioma)

Esses dados não são enviados para nenhum servidor externo, a menos que você opte voluntariamente pela funcionalidade de backup na nuvem.

☁️ Dados de backup na nuvem Opcional

Se você optar por usar a funcionalidade de sincronização com Google Drive, o aplicativo irá:

  • Solicitar autenticação via Google Sign-In
  • Acessar seu Google Drive para criar, ler e atualizar arquivos de backup do Lembre+
  • Armazenar os backups na pasta "LembrePlus Backups" no seu Google Drive (ou no espaço de dados do aplicativo)

O escopo de acesso utilizado é drive.file, que limita o acesso apenas aos arquivos criados pelo próprio aplicativo. Não acessamos outros arquivos do seu Google Drive.

2. Informações que NÃO coletamos

O Lembre+ não coleta, armazena nem compartilha:

  • Dados pessoais identificáveis (nome, e-mail, telefone) — exceto a conta Google usada para login voluntário
  • Dados de localização (GPS)
  • Dados de uso, analytics ou telemetria
  • Informações financeiras ou de pagamento
  • Dados de contatos, fotos, câmera ou microfone
  • Identificadores de publicidade

3. Permissões do aplicativo

O Lembre+ solicita as seguintes permissões do Android:

  • INTERNET — Necessária para autenticação com Google e backup na nuvem
  • POST_NOTIFICATIONS — Para enviar notificações de lembretes
  • SCHEDULE_EXACT_ALARM — Para agendar alertas precisos
  • RECEIVE_BOOT_COMPLETED — Para reagendar notificações após reiniciar o dispositivo
  • VIBRATE — Para vibrar o dispositivo ao receber notificações

Todas as permissões são utilizadas exclusivamente para as funcionalidades descritas e nenhum dado é transmitido a terceiros.

4. Google Sign-In e Google Drive

Quando você utiliza a funcionalidade de backup na nuvem:

  • A autenticação é realizada diretamente com o Google, seguindo as políticas de privacidade do Google
  • O Lembre+ recebe apenas o token de autenticação para acessar o Google Drive
  • Não armazenamos sua senha do Google
  • Você pode revogar o acesso a qualquer momento nas configurações da sua conta Google

5. Compartilhamento de dados

O Lembre+ não compartilha seus dados com terceiros. Especificamente:

  • Não vendemos dados pessoais
  • Não utilizamos serviços de analytics
  • Não exibimos anúncios
  • Não integramos SDKs de rastreamento

6. Segurança dos dados

Implementamos as seguintes medidas de segurança:

  • Os dados são armazenados localmente no dispositivo, protegidos pelo sistema operacional Android
  • A comunicação com o Google Drive utiliza conexões seguras (HTTPS/TLS)
  • O acesso ao Google Drive é limitado pelo escopo drive.file
  • O aplicativo utiliza ofuscação de código (ProGuard) na versão de produção

7. Retenção e exclusão de dados

  • Dados locais: Permanecem no dispositivo até que você os exclua manualmente ou desinstale o aplicativo
  • Backups na nuvem: Permanecem no seu Google Drive até que você os exclua manualmente
  • Desinstalação: Ao desinstalar o Lembre+, todos os dados locais são removidos automaticamente pelo Android

8. Crianças

O Lembre+ não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores. Se você acredita que uma criança nos forneceu dados pessoais, entre em contato conosco.

9. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Alterações significativas serão comunicadas através de atualizações do aplicativo. Recomendamos revisar esta página regularmente.
